# ⚡ CAGO Converter

[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)  
[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://your-netlify-link.netlify.app)

**CAGO Converter** is a modern web application that allows users to convert PDF files into **TXT** and **Word (.docx)** formats seamlessly. The app features a **glassmorphic UI**, **animated gradients**, and a **login system** for a professional website-like experience.

---

## 🌟 Features


- **PDF to TXT**: Extract text from PDF files and download as `.txt`.
- **PDF to Word**: Convert PDF text to `.docx`.
- **Glassmorphic UI**: Smooth animations with neon glow and modern design.
- **Responsive & Interactive**: Works on both desktop and mobile.
- **Branded Footer**: Made by CODIXCO, Owner: W.C Maleesha De Silva.

---

## 🚀 How to Use

1. Open the site.
2. Upload a PDF file.
3. Click **Convert** to extract text.
4. Download as **TXT** or **Word**.

---

## 🛠 Built With

- HTML, CSS, JavaScript
- [pdf.js](https://mozilla.github.io/pdf.js/) for PDF parsing
- [docx.js](https://www.npmjs.com/package/docx) for Word file generation
- Glassmorphism & CSS animations
